# 機能一覧
▼ソーシャル機能
	ユーザー登録
	ログイン、ログアウト
	メール認証によるパスワードリセット
	ユーザー検索
	ランキング
	他ユーザーのフォロー
	他ユーザーの学習時間、日記のタイムライン
▼学習機能
	マーカー登録機能
	タイマーによる学習時間記述
	方眼紙による学習時間記述
	今月、今年の学習時間のグラフ
	今までのぬり絵シート表示
	ぬり絵シートダウンロード
	学習分野別の勉強時間表示
	過去の学習履歴表示
	日記を書く機能
	To-doリスト機能
